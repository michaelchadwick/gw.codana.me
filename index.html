Gem Warrior

Gem Warrior is a text adventure created by Michael Chadwick in Ruby, packaged as a Ruby Gem.

It can be installed in two steps, as long as you have Ruby and RubyGems installed:

  1. gem install gemwarrior
  2. gemwarrior

More information can be found on Github
