Gem Warrior

Gem Warrior is a text adventure created by Michael Chadwick in Ruby, packaged as a RubyGem.

It can be installed in two steps, as long as you have Ruby and RubyGems installed:

  1. (sudo) gem install gemwarrior
  2. gemwarrior

Source can be found on Github

Make sure you have the latest version by typing (sudo) gem update
